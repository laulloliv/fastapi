# API REST com FastAPI e DOCKER

Uma API Rest para Lista de Tarefas

## Features:
- Create, Read, Update and Delete Task

## Tecnologias:
- Python/FastAPI
- Docker
- MongoDB

## Instalação com Docker

```cpp
docker-compose up --build
```

## Instalação sem Docker

```cpp
pip install -r requirements.txt
```


```cpp
python main.py
```
